# Joke Generator 🎉

A fun and interactive web application that generates random jokes at the click of a button!

## Features

- 🎲 **Random Joke Generation** - Get a new joke every time
- 💾 **Joke History** - Keep track of your favorite jokes
- 🎨 **Clean UI** - Simple and intuitive interface
- ⚡ **Fast & Responsive** - Works seamlessly on all devices

## Demo

Visit the live application: [Joke Generator](https://djake64.github.io/joke-generator/)

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required!

### Usage
1. Open the application in your browser
2. Click the "Get Joke" button
3. Enjoy the joke!
4. Repeat for more laughs 😄

## Technologies Used

- CSS3

## How It Works

The joke generator uses a collection of pre-loaded jokes and randomly selects one each time you click the button. No external API calls required!
